## 大鱼TV
* 主要包括电视台直播、电影，地方台，央视，卫视，网络台等，将近700多个电视台。
* 发表仿微博
* 发现，搜索
* 会员登录
* 评论，自适应高度，多级评论

### 引用第三方技术包括：
* ijkplayer
* sharesdk分享，登录
* Masonry
* zfplayer等
