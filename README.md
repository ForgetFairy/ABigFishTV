## 大鱼TV
* 主要包括电视台直播、电影，地方台，央视，卫视，网络台等，将近700多个电视台。
* 发表微博
* 发现，搜索
* 会员登录
* 评论

### 引用第三方技术包括：
* ijkplayer
* sharesdk分享，登录
* Masonry
* zfplayer等

### 后台服务采用THINKPHP5写的，
* 地址：<a href="https://github.com/clyhs/webapi">webapi</a>

### 更换APP服务地址
* 在ABigFishTV/Foundation/Common/ABigFishHeader.h的 #define BaseUrl @"http://webapi.abigfish.org"
